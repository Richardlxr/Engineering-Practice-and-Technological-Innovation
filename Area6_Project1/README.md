# Area6 Project - Mapping SJTU

## 文件夹说明

- `Buildings_Area6/` - 建筑物矢量数据（Polygon）
- `Roads_Area6/` - 道路矢量数据（Line）
- `Grassland_Area6/` - 草地/绿化矢量数据（Polygon）
- `PoI_Area6/` - 兴趣点矢量数据（Point）
- `Area6.qgz` - QGIS 项目文件（从 QGIS 另存为到此目录）

## 工具

- QGIS 3.x
- EPSG:4326 (WGS 84)
